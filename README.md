# 🌍 AI-Based Trip Planner and Recommendation System

An AI-powered web application that generates personalized travel itineraries and destination recommendations based on user preferences. The system leverages Google Gemini AI to create intelligent travel plans and provides weather information, interactive maps, and hotel recommendations for a seamless trip planning experience.

## ✨ Features

- 🤖 AI-powered trip itinerary generation using Google Gemini
- 📍 Personalized destination recommendations
- 🗺️ Interactive maps with Folium & OpenStreetMap
- 🌤️ Real-time weather information
- 🏨 Hotel recommendations
- 👤 User Authentication (Login & Registration)
- 💬 User Feedback System
- 📱 Responsive User Interface

## ⚙️ Installation

### 1. Clone the repository
```bash
git clone https://github.com/your-username/ai-trip-planner-and-recommendation-system.git
```

### 2. Navigate to the project
```bash
cd ai-trip-planner-and-recommendation-system
```

### 3. Create a virtual environment
```bash
python -m venv venv
```

### 4. Activate the virtual environment

**Windows**
```bash
venv\Scripts\activate
```

**Linux / macOS**
```bash
source venv/bin/activate
```

### 5. Install dependencies
```bash
pip install -r requirements.txt
```

### 6. Apply migrations
```bash
python manage.py migrate
```

### 7. Run the server
```bash
python manage.py runserver
```

Open your browser and visit:
http://127.0.0.1:8000/
